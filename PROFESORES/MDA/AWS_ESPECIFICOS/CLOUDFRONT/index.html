Cloud Engineering Training

Instructor: Pedro Nieto

Welcome to the Cloud Engineering Training session. This training is designed to provide you with the knowledge and skills required to become a proficient Cloud Engineer. Throughout this course, you will learn about various cloud technologies, best practices, and hands-on techniques to manage and deploy cloud infrastructure.

Course Outline:

  • Introduction to Cloud Computing
  • Cloud Service Models (IaaS, PaaS, SaaS)
  • Cloud Deployment Models (Public, Private, Hybrid)
  • Cloud Providers Overview (AWS, Azure, Google Cloud)
  • Setting Up Cloud Infrastructure
  • Managing Cloud Resources
  • Security and Compliance in the Cloud
  • Cost Management and Optimization
  • Hands-on Labs and Projects

About the Instructor:

Pedro Nieto is a seasoned Cloud Engineer with extensive experience in designing, deploying, and managing cloud infrastructure. He has worked with various cloud platforms and has a deep understanding of cloud technologies and best practices. Pedro is passionate about teaching and helping others succeed in their cloud engineering careers.
